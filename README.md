## 📌 Project 2: xNote (Backend)

### 🔹 Overview

**xNote** is a backend REST API project that allows users to manage notes with full CRUD functionality.

### 🔹 Tech Stack

* **Node.js**
* **Express.js**
* **MongoDB**

### 🔹 Core Features

* ✍️ Create notes
* 📖 Retrieve notes
* ✏️ Update notes
* ❌ Delete notes

### 🔹 API Methods Used

* `GET` – Fetch data
* `POST` – Create data
* `PATCH` – Update data
* `DELETE` – Remove data

### 🔹 Concepts Implemented

* REST API design
* HTTP protocol handling
* Middleware usage
* MongoDB schema & models
* Package/module management (npm)
* Server-client data flow

### 🔹 Learning Outcome

* Understood how backend systems work
* Built scalable API structure
* Learned how data is processed and served to clients
